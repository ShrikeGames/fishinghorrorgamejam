# Fishing Horror Game Jam 2

I'll be creating a fishing horror game in 14 days. 
Fishing has to be the core mechanic, it must be horror themed, and include a story with some npcs.

Hosted by Figerox Studios
The rules and FAQ about the game jam itself are here: https://itch.io/jam/fishing-horror-jam-2

# Rules

- Fishing must be a main gameplay mechanic, not just a side activity.
- The game must lean into horror.
- Include a story and some NPCs.
- Any engine or tools are allowed.
- Solo or team entries are welcome.
- Reused code and assets are fine, but games must be created during the jam. 

# The voting Categories are:

- Fishing Enjoyment
- Fishing Beastiary
- Horror Elements
- Lore
- Visuals and Audio

# Optional Theme

- The (completely optional) secret theme for the fishing horror jam is...
- "A Bad Feeling".

# Endings
- 1/6 Starve to death
- 2/6 Your corruption bar fills
- 3/6 Cat score is too high or you give him an angel fish
- 4/6 Dog score is too high or you give him an angel fish
- 5/6 Catch a demon fish in the deep (it catches you more like)
- 6/6 Catch and eat the angel fish
